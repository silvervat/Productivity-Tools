4️⃣ JÄRGMISED SAMMUD

A. LOKAALNE TESTIMINE

   1. Terminal: cd Productivity-Tools
   2. Terminal: npm install
   3. Terminal: npm run dev
   4. Browser: http://localhost:5173
   5. Kontrolli et komponendid kuvatakse ✅

B. TRIMBLE CONNECT'IGA TESTIMINE

   1. Ava https://connect.trimble.com
   2. Oma projekt → Laiendused
   3. ➕ Upload Custom Extension
   4. Vali: public/extension.json
   5. Dev URL: http://localhost:5173
   6. Load → Kõik käib! ✅

C. GITHUBIS TÕUSMINE

   1. GitHub repo loomine (New repository)
   2. Terminal: git remote set-url origin <your-repo>
   3. Terminal: git push -u origin main
   4. ✅ Kõik faild on GitHubis!

D. TOOTMISSE KÄIVITAMINE (hiljem)

   1. Terminal: npm run build
   2. Laadi dist/ Vercel/Azure/Netlify
   3. Update extension.json pluginUrl
   4. ✅ Production käib!
